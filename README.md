# Mendak60
A 3D printable 60% keyboard based on the frog mini by Geonworks.
Uses the frog mini plate and tadpoles.
Bottom mount
6.9* typing angle
~21mm front height
![Photo1](photo.png?raw=true)

Required Hardware:
1. 6x M3 Heat set inserts.
2. 6x M3x10 socket head hex screws.
3. 8mm bumpons/feet.
4. Frog mini plate.
5. 8x Geon tadpoles.
